## Task 1: Web Scraping :  Highest Grossing Movies Web Scraper
### Website: https://en.wikipedia.org/wiki/List_of_highest-grossing_films
## Project Overview

This project demonstrates Web Scraping using Python.

The program extracts movie data from Wikipedia and creates a structured CSV dataset.

## Technologies Used

- Python
- Requests
- BeautifulSoup
- Pandas

## What I did:
- Collected movie data from the Wikipedia page of the highest-grossing films using Python.
- Used the **Requests** library to download the webpage and **BeautifulSoup** to parse the HTML content.
- Extracted key information including **Rank, Movie Title, Worldwide Gross, and Release Year** from the HTML table.
- Organized the extracted data into a structured **Pandas DataFrame** and exported it as a CSV dataset (`highest_grossing_movies.csv`).
- Learned the fundamentals of web scraping, HTML structure, and data extraction while creating a reusable dataset for future data analysis.

## Output
highest_grossing_movies.csv
