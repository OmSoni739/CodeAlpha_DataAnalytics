# 🎬 Highest Grossing Movies Web Scraper

## Project Overview

This project demonstrates Web Scraping using Python.

The program extracts movie data from Wikipedia and creates a structured CSV dataset.

## Technologies Used

- Python
- Requests
- BeautifulSoup
- Pandas

## Dataset

The scraper collects:

- Rank
- Movie Name
- Worldwide Gross
- Release Year

## Output

highest_grossing_movies.csv

## How to Run

pip install -r requirements.txt

python scraper.py
